# Design — índice

Todo lo que define **qué** es el juego y **cómo** debería sentirse, sin entrar en implementación C++.

| Carpeta | Uso |
|---------|-----|
| **GDD/** | Game Design Document y documentos de diseño de juego. Punto de entrada: [GDD/GAME_DESIGN_DOCUMENT.md](GDD/GAME_DESIGN_DOCUMENT.md). |
| **Lore/** | Narrativa, mundo, personajes, referencias para escritura y misión. |
| **Art/** | Briefs, moodboards, referencias visuales, notas de dirección de arte. |
| **Audio/** | Dirección de sonido, referencias musicales, notas de SFX. |

Para **estructura técnica del repo** y rutas de código, ver el [README](../README.md) en la raíz del proyecto y [Docs/ONBOARDING.md](../Docs/ONBOARDING.md).

Para **contribuir** (flujo Git, documentación, PRs): [CONTRIBUTING.md](../CONTRIBUTING.md).
