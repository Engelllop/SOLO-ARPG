# 99 — Glosario in-universe

> Referencia rápida para escritores. **Toda quest, diálogo de NPC, descripción de ítem y placa ambiental debe usar exclusivamente términos de este glosario.** Los términos meta-canon (Compilación, Patrón, Borrador, etc.) **nunca aparecen en boca de personaje hasta el Acto V**.

---

## Términos cosmológicos

| In-universe | Significado real | Quién lo usa |
|---|---|---|
| **El Velo** | El mundo material, el Borrador | Todos |
| **La Tierra Provisional** | Sinónimo formal de El Velo | Sabios, eruditos |
| **El Tejido** | El sustrato cósmico donde se proyecta el Velo | Sabios, oráculos |
| **El Patrón** | El plano "perfecto" del Arquitecto | Templarios, herejes |
| **El Modelo** | Sinónimo enano de El Patrón | Enanos, archivistas |
| **La Última Mano** | La compilación final del mundo (mítica, esperada) | Religiones populares |
| **La Obra** | Sinónimo poético de la Compilación | Bardos |
| **Las Edades Olvidadas** | Iteraciones anteriores del Borrador | Todos los que las nombran |
| **Los Mundos Caídos** | Sinónimo dramático de Las Edades Olvidadas | Bardos, sacerdotes |
| **Esquirla** | Fragmento que persiste entre Edades | Sabios, anticuarios |
| **Reliquia del Tejido** | Esquirla particularmente potente | Sabios |
| **Persistente** | Hilo (alma) que sobrevive a la recompilación | Casi nadie usa esta palabra; los que la usan son herejes |
| **Hilo** | Un alma | Sabios, oráculos |
| **Ser Recogido** | Morir | Todos. Eufemismo común |
| **El Tejedor** | El Arquitecto | Templarios, herejes |
| **El Modelador** | Sinónimo de El Tejedor | Sabios élficos |
| **Aquel que Pulsa el Telar** | Sinónimo poético | Bardos |
| **El Que Borra** | El Arquitecto, en boca de los herejes que le temen | Demonios, paranoicos |

## Frases hechas / refranes del mundo

Para sembrar como ambient dialogue:

- *"Si miras un mapa el tiempo suficiente, los ríos cambian de cauce."* (cartógrafos)
- *"La Aguja recuerda."* (cierre habitual de cuentos)
- *"Bébelo despacio. La memoria es lenta."* (taberneros)
- *"Quien clava la primera espada no es el más fuerte. Es el primero en mirar."* (gremios)
- *"Tirenne ya no existe. Eso no significa que no estuviera."* (refrán de Sahal)
- *"Reza si quieres, pero no esperes respuesta. Las respuestas vienen sin oración."* (Vassen, generalizable)
- *"El Velo se rasga donde nadie mira."* (sabios, raro)
- *"No nombres a los que no están. Los llamas."* (madres a sus hijos)
- *"Tres cosas no se compran: el sueño, el silencio, y el primer paso."* (mercaderes)
- *"La Aguja se abre cuando alguien recuerda."* (proverbio críptico, mencionado por el anciano demente de Tirenne)

## Lo que **nunca** se dice in-universe (hasta el Acto V)

Lista negra. **Si una de estas palabras aparece en un diálogo del Acto I–IV, está mal escrito y hay que reescribir:**

- "Compilar" / "Compilación"
- "Borrador"
- "Iteración"
- "Reescritura"
- "Build"
- "Diff"
- "Merge"
- "Resolución" (en sentido informático)
- "Patch"
- "Variable"
- "Bug"
- "Sistema"
- "Programa"
- "Código"
- "Algoritmo"
- "Simulación"
- "Realidad virtual"
- "Universo"

> Estas palabras se reservan para el Acto V, cuando el Arquitecto y los Heraldos Caídos las pronuncian directamente. Su irrupción es **el síntoma narrativo** de que el velo se ha roto.

## Plantilla rápida: cómo traducir un concepto al idioma del mundo

Si necesitas que un personaje hable de un concepto cósmico, usa esta tabla mental:

| Quieres decir... | Di mejor... |
|---|---|
| "Bug" / "error del sistema" | *"Una grieta en el Velo"*, *"un descosido del Tejido"* |
| "Reset" / "recompilar" | *"La Última Mano"*, *"el siguiente vuelco"*, *"cuando el Tejedor levante el telar"* |
| "Save point" | *"Asilo del Tejido"* (formal), *"un lugar donde el Hilo se quieta"* (poético) |
| "NPC" | (no aplica, no debería existir el concepto) |
| "Quest" | *"un encargo"*, *"un recado"*, *"una palabra dada"* |
| "Loot" | *"despojos"*, *"lo que cae de las manos del muerto"*, *"hallazgo"* |
| "Stats / atributos" | *"el temple"*, *"el peso del cuerpo"*, *"la herencia"* |
| "Level up" | *"madurar"*, *"asentarse"*, *"ganarse otro nombre"* (cuando se ganan títulos) |
| "Respawn" | *"despertar"*, *"el Tejedor te trajo de vuelta"* (sospechoso si lo dicen demasiado) |

## Nota final para escritores

El glosario es vivo. Si encuentras una palabra que necesitas usar y no está aquí, **inventa una versión in-universe y agrégala a este archivo**. El criterio de aceptación es:

1. Suena medieval-fantástica, no técnica.
2. Es decible en voz alta sin trabarse.
3. Tiene resonancia con los términos ya existentes (preferimos *El Velo*, *El Tejido*, *El Patrón* — léxico de tela y manualidad — sobre vocabulario abstracto).

Cuando dudes, **consulta este archivo antes de escribir un diálogo**. La consistencia léxica es la mitad de la magia del worldbuilding.
